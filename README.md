# Home Renovation PWA
This is the repository for the Home Renovation PWA to track and manage renovations around the house.

**Live (GitHub Pages):** https://wildurbear.github.io/HR/ — open on your phone and use *Add to Home Screen* to install.

Currently in the design-mockup phase: front-end only, in-memory placeholder data, no backend yet. See [CLAUDE.md](CLAUDE.md) for the full project brief.
